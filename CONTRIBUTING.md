# Contribuir al Sistema de Ventas

- Ramas: feature/..., test/..., ci/..., fix/..., docs/...
- Commits: feat:, test:, ci:, fix:, docs:
- Abre PR hacia `main` y completa la plantilla.
- Backend: `composer lint`, `composer static`, `php artisan test`.
- Frontend: `npm ci`, `npm run lint`, `npm run test`, `npm run build`.
- No subas credenciales ni archivos `.env`.
- Respeta `.editorconfig` y estilos del repo.
- Describe cambios, evidencia y riesgo/rollback en el PR.
- Mantén cambios pequeños y enfocados.
